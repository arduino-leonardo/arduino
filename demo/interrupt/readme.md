# 按键控制 LED (中断方式)

## 功能概述

通过外部中断实现异步的按键捕捉。

- 通过数字引脚 3 捕捉外部中断 0 信号
- 通过数字引脚 13 控制板载的 LED 灯 L

按键按下，LED 状态反转一次。
